## Tech-Tracker Server :snowflake:

### 1. Pull Repo
```
$ git clone https://github.com/jakeshoemaker/tech-tracker-server.git
```
### 2. Install dependencies
``` 
$ yarn || npm install
```
### 3. compile typescript && run server
```
$ yarn tsc
$ yarn dev
```
